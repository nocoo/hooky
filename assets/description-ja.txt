Hooky — ワンクリックで Webhook を送信、ページコンテキストを自動取得、スマートルール対応。

Webhook テンプレートを作成し、ページデータを自動入力して、ツールバー、右クリックメニュー、またはクイック送信ルールからワンクリックで送信。

🪝 WEBHOOK テンプレート
複数の名前付き Webhook テンプレートを作成。URL、HTTP メソッド（GET/POST/PUT/PATCH/DELETE）、キー・バリューパラメータをそれぞれ個別に設定可能。

🔀 テンプレート変数
テンプレート変数でリアルタイムのページ情報を Webhook パラメータに挿入：
• {{page.url}} — 現在のページ URL
• {{page.title}} — ページタイトル
• {{page.selection}} — 選択テキスト
• {{page.meta.description}} — Meta description
• {{page.meta.og:title}} — Open Graph タイトル
• {{page.meta.og:description}} — Open Graph 説明
• {{page.meta.og:image}} — Open Graph 画像 URL

⚡ クイック送信ルール
ルールを定義して、現在のページに基づいて送信する Webhook テンプレートを自動選択。ツールバーアイコンをクリックすると、ルールが順番に評価され、最初にマッチしたルールの Webhook が即座に送信されます — ポップアップは開きません。

ルール条件：URL またはタイトル × 含む、一致、前方一致、後方一致、正規表現マッチ。優先順位の並べ替え、個別の有効/無効が可能。どのルールにもマッチしない場合は、通常通りポップアップが開きます。

バッジフィードバック：✓（緑）送信成功、✗（赤）送信失敗。

📋 コンテキストメニュー
任意のページで右クリックすると「Hooky」メニューにすべてのテンプレートが表示され、ワンクリックで送信。

🎨 テーマ
システム、ライト、ダークの 3 種類から選択。

🌐 言語
10 言語対応：English、简体中文、繁體中文、日本語、한국어、Français、Deutsch、Español、Português (BR)、Русский。

🔒 プライバシー
Hooky はユーザーデータの収集・保存・送信を一切行いません。設定はすべてブラウザのローカル（chrome.storage.local）に保存されます。Webhook リクエストはユーザーが設定した URL にのみ送信されます。

ユースケース
• 記事を Notion、Raindrop、または任意の API に保存
• ページ URL を Slack、Discord、Telegram ボットに送信
• ページメタデータで CI/CD パイプラインをトリガー
• ブラウジング履歴をパーソナル Webhook に記録
• ページをカスタムバックエンドにブックマーク

権限
• activeTab — テンプレート変数用に現在のページ URL とタイトルを取得
• storage — Webhook テンプレートとルールをローカルに保存
• contextMenus — 右クリックメニューに「Hooky」を追加
• ホスト権限（すべての URL）— 設定した任意の URL に Webhook リクエストを送信可能にします。ほとんどの Webhook エンドポイントはサーバー間通信用に設計されており、ブラウザの CORS プリフライトリクエストを処理しません。この権限がないと、JSON ボディを含む POST リクエストがブラウザによってブロックされます。Hooky は明示的に設定された URL にのみリクエストを送信します
